# 客货一体化农村公交AI协同优化系统

## 🚌 项目简介

本项目实现了**基于大语言模型(LLM) AI协同的乘客-配送客户协同优化系统**，专门针对农村客货邮融合的公交运输场景。

### 核心特点

- ✅ **AI智能协同**：模拟大语言模型进行智能决策，动态平衡乘客和货物需求
- ✅ **多目标优化**：同时优化成本、时间、服务质量等多个目标
- ✅ **以人为本**：将乘客体验和配送满意度作为核心优化指标
- ✅ **完整实现**：包含数学模型、优化算法、可视化工具

---

## 📚 研究背景

### 问题背景
农村地区面临：
- 人口分散，公交运营成本高
- 物流配送困难，快递进村难
- 车辆利用率低，资源浪费严重

### 解决方案
**客货一体化**：公交车在运送乘客的同时配送货物
- 提高车辆利用率
- 降低物流成本
- 改善农村服务

### 创新点
**AI协同优化**：利用大语言模型智能协调乘客和货物需求
- 动态权重调整
- 智能推理决策
- 实时自适应优化

```python
# AI引擎根据实时情况调整优先级
if passenger_wait_time > 30:
    alpha = min(0.8, alpha + 0.2)  # 提高乘客优先级
elif cargo_urgency > 0.8:
    beta = min(0.6, beta + 0.2)  # 提高货物优先级
```

```
min Z = w₁·成本 + w₂·(1-乘客满意度) + w₃·(1-货物满意度)
```
